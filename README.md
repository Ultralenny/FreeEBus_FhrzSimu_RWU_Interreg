Die ist eine Fahrzeugsimulation eines Elektro Bus, im Rahmen des Interreg Projekts FreeE-Bus.  
Das Programm wurde in Python programmiert und berechte itterativ alle Fahrwiderstände über eine Vorgegebene Strecke.

Der Ordner "Data" enthält das Fahrprofil und die verwendeten Daten für das Fahrzeugmodell
Der Ordner "Src" enhält alle Python-Dateien

Die Fahrzeugsimulation wird über die Datei main.py im Ordner "Src" gestartet und berarbeite itterativ das Fahrprofil. 
in Config_Data können die Fahrzeugdaten angepasst werden.
